# fichiers-prénoms

## Cloner le projet
Par commande:
git clone https://github.com/Poluux/fichiers-prenoms.git
cd fichier-prenoms

## Créer chacun une branche
Par commande:
git checkout -b ajout-prenom-"prenom"
